BareBrowser

BareBrowser understands the following internal URLs.
Internal URLs can be used in the address bar and on the command-line.

bb://home, home: Go to internal home page (this page).

bb://readme, readme: Show README file (HTML format).

bb://readme.md, readme.md: Show README file (Markdown format).

bb://settings, settings: Show current settings (may differ from the content of settings.json).

bb://info, info: Show product, installation and system info.

bb://license, license: Show software license.

bb://changes, changes: Show changelog.


Navigation (even on the command-line) is also possible with:

bb://reload, reload: Reload the currently loaded page.

bb://back, back: Go one step back in the browser history.

bb://forward, forward: Go one step forward in the browser history.

bb://close, close: Close a window.
